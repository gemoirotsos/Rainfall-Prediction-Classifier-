# Rainfall Prediction using Machine Learning

This project focuses on building a machine learning pipeline to predict rainfall using a real-world weather dataset (Australian Weather Dataset).

## Overview

The goal of this project is to analyze meteorological data and develop classification models that can predict whether it will rain the next day based on historical weather features.

## Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Dataset

- Australian Weather Dataset
- Includes meteorological features such as temperature, humidity, wind, pressure, and rainfall indicators

## Methodology

### Data Preprocessing
- Handled missing values and cleaned the dataset
- Performed feature selection and transformation
- Encoded categorical variables where necessary

### Feature Engineering
- Prepared relevant meteorological variables for model training
- Structured the dataset for supervised learning

### Model Development
- Implemented and compared:
  - Logistic Regression
  - Random Forest

### Model Optimization
- Applied Grid Search Cross-Validation to tune hyperparameters
- Selected optimal model configurations based on performance

### Evaluation
- Evaluated models using classification metrics such as:
  - Accuracy
  - Precision
  - Recall
  - F1-score

- Visualized results using Matplotlib

## Results

The project demonstrates how different machine learning models perform on real-world weather data and highlights the importance of feature engineering and model tuning in improving prediction accuracy.
